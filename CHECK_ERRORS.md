# エラーの確認方法

## 開発者ツールでエラーを確認する手順

### Step 1: Consoleタブを開く

1. 開発者ツール（F12）が開いていることを確認
2. 上部のタブから「Console」をクリック
3. 赤いエラーメッセージがないか確認

### Step 2: エラーメッセージを確認

Consoleタブに表示される可能性のあるエラー：

1. **"Failed to fetch"**
   - Supabaseへの接続エラー
   - 環境変数が正しく設定されていない可能性

2. **"Invalid supabaseUrl"**
   - `.env.local`の値が正しくない

3. **"Cannot read property of undefined"**
   - コンポーネントのエラー

4. **"Module not found"**
   - 依存関係の問題

### Step 3: Networkタブを確認

1. 「Network」タブをクリック
2. ページをリロード（F5またはCmd+R）
3. 赤いエラー（4xx、5xx）がないか確認
4. 特に `/api/` や Supabase へのリクエストが失敗していないか確認

### Step 4: エラーメッセージをコピー

1. Consoleタブでエラーメッセージを右クリック
2. 「Copy」を選択
3. エラーメッセージを共有してください

## よくある問題

### 白い画面が表示される

**確認事項**:
- Consoleタブにエラーが表示されているか
- Networkタブでリクエストが失敗していないか

### ページが読み込まれない

**確認事項**:
- ターミナルで開発サーバーが起動しているか
- `http://localhost:3000` にアクセスしているか

### 認証エラー

**確認事項**:
- `.env.local`が正しく設定されているか
- SupabaseでGoogle認証が有効になっているか
